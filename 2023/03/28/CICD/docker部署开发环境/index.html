部署流程

  1. 购买服务器
  2. 安装docker
  3. 部署后端
    1. 准备后端基础运行环境
      1. jdk
      2. redis
      3. mysql
        1. 拉镜像
        2. 启动mysql
        3. 服务器开放对外mysql端口
        4. navicat连接服务器，执行初始化sql脚本
    2. 制作后端应用镜像
      1. maven install 打包
      2. 编写Docfile
      3. 两者都上传到服务器同一目录下
      4. 进入该目录，制作镜像
    3. 后端启动脚本
      1. 脚本内容
      2. 编写为应用启动脚本
    4. 启动后端应用
    5. 验证
      1. 查看启动日志
      2. 开放服务器端口号
      3. PostMan测试接口
      4. 关闭服务器端口号
  4. 部署前端
    1. 前端基础运行环境
    2. 前端代码打包
    3. 配置nginx
    4. 启动nginx
