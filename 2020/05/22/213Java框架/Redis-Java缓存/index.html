  • @Cacheable - 表明对应方法的返回结果可以被缓存，首次调用后，下次就从缓存中读取结果，方法不会再被执行了。

  • @CachePut - 更新缓存，方法每次都会执行

  • @CacheEvict - 清除缓存，方法每次都会执行
